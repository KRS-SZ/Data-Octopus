==============================================
  DATA OCTOPUS - Installation Guide
  Version 5.2.0
==============================================

Willkommen bei Data Octopus!
Professional STDF Wafermap Analyzer mit Manifold Integration.

----------------------------------------------
WINDOWS INSTALLATION (Empfohlen)
----------------------------------------------

1. Den gesamten "Windows" Ordner kopieren
2. "start.bat" doppelklicken
   → DataOctopus.exe wird gestartet

Das war's! Keine Python-Installation nötig.

HINWEIS: Der erste Start kann 10-30 Sekunden dauern.


----------------------------------------------
MAC / LINUX INSTALLATION
----------------------------------------------

Voraussetzungen:
- Python 3.10 oder höher
- tkinter (für GUI)

Installation:

1. Terminal öffnen
2. In den Mac_Linux Ordner wechseln:
   cd /pfad/zu/Mac_Linux

3. Installer ausführen:
   chmod +x install.sh
   ./install.sh

4. App starten:
   ./run.sh


----------------------------------------------
MANUELLE INSTALLATION (alle Plattformen)
----------------------------------------------

1. Python 3.10+ installieren (mit tkinter!)

2. Dependencies installieren:
   pip install -r requirements.txt

3. App starten:
   python main_v5.1.py
